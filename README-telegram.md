# راهنمای دریافت خودکار نرخ طلا از کانال تلگرام اتحادیه

سایت همین الان هم کار می‌کند: از پنل ادمین → تب «نرخ طلا و تلگرام» می‌توانید
نرخ را **دستی** وارد کنید و همه‌ی قیمت‌های سایت آنی به‌روز می‌شوند. این بخش
فقط برای **خودکار کردن** همان کار است.

## گام ۱: ساخت ربات تلگرام
1. در تلگرام به `@BotFather` پیام بدهید.
2. دستور `/newbot` را بزنید.
3. یک نام و یک یوزرنیم (باید به `bot` ختم شود) برای ربات انتخاب کنید.
4. BotFather یک **توکن** به شما می‌دهد، شبیه: `123456789:AAExampleToken`.
   این توکن را جایی امن نگه دارید.

## گام ۲: عضو کردن ربات در کانال
1. وارد کانال تلگرام اتحادیه بشوید (لینکش را برای من بفرستید تا در راهنما
   دقیق‌تر کمک کنم؛ اما مراحل زیر برای هر کانالی یکسان است).
2. اگر ادمین کانال هستید، ربات را به‌عنوان عضو یا ادمین کانال اضافه کنید.
   اگر ادمین کانال نیستید، باید از مدیر کانال بخواهید ربات را اضافه کند —
   چون تلگرام اجازه نمی‌دهد یک ربات بدون اجازه‌ی مدیر، پیام‌های کانال را
   بخواند.
3. یوزرنیم کانال (مثل `@goldunion`) یا آیدی عددی آن را یادداشت کنید.

## گام ۳: دو راه برای اتصال

### راه ساده (مستقیم از مرورگر - ممکن است با خطای CORS مواجه شود)
در پنل ادمین → «نرخ طلا و تلگرام»، توکن ربات و آیدی کانال را وارد و روی
«دریافت نرخ الان» کلیک کنید. بعضی هاست‌ها/مرورگرها اجازه‌ی این تماس مستقیم
را می‌دهند؛ اگر خطا دیدید، از راه مطمئن زیر استفاده کنید.

### راه مطمئن (توصیه‌شده): اسکریپت سمت سرور + Cron Job
1. فایل `server/fetch_rate.php` را باز کنید و مقادیر `$BOT_TOKEN` و
   `$CHANNEL` را با اطلاعات واقعی خودتان پر کنید.
2. این فایل را روی هاست سایت آپلود کنید.
3. از پنل هاست (سی‌پنل یا مشابه)، یک Cron Job بسازید که هر ۵ دقیقه این
   دستور را اجرا کند:
   ```
   */5 * * * * php /home/username/public_html/server/fetch_rate.php
   ```
4. این اسکریپت فایل `server/rate.json` را می‌سازد و به‌روز نگه می‌دارد.
5. در نهایت باید تنظیمات فروشگاه (بخش ادمین) به‌جای خواندن نرخ از
   localStorage، این فایل JSON را از سرور بخواند — این تغییر وقتی سایت را
   به سرور واقعی وصل کردیم (طبق درخواست شما) انجام می‌شود.

## نکته‌ی امنیتی
توکن ربات را هرگز در کد سمت مرورگر (فایل‌های html/js قابل مشاهده برای همه)
قرار ندهید؛ فقط در فایل PHP سمت سرور نگه دارید، چون هرکسی که آن را ببیند
می‌تواند از ربات شما سوءاستفاده کند.
